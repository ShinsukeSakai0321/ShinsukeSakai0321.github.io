PYTHON用のユーティリティパッケージのインストール手順を示します

  • ターミナルモードで次のコマンドにより，パッケージをGitHubからインストールします
    pip install git+https://github.com/ShinsukeSakai0321/Utility
  • パッケージの更新をするときには次のコマンドを入力します(初回は不要)
    pip install --upgrade git+https://github.com/ShinsukeSakai0321/Utility
  • プログラム内でパッケージ内のモジュールを使う時，以下のimport文を記載する
    from Utility import モジュール名 as 参照名
    例えば，単位変換モジュールを利用するときには，下記を入力する．
    from Utility import UnitConv as uc
