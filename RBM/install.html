• ターミナルモードで次のコマンドにより，パッケージをGitHubからインストールします
    pip install git+https://github.com/ShinsukeSakai0321/RBM
  • パッケージの更新をするときには次のコマンドを入力します(初回は不要)
    pip install --upgrade git+https://github.com/ShinsukeSakai0321/RBM
  • プログラム内でパッケージ内のモジュールを使う時，以下のimport文を記載する
    from RBM import モジュール名 as 参照名
    例えば，信頼性関係のモジュールを利用するときには，下記を入力する．
    from RBM import reliability as rel
